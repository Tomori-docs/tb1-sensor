**1. Which of the following statements correctly distinguishes a feedforward system from a feedback system? ***

A. A feedforward system continuously compares the output with the desired value and corrects the error.

B. A feedforward system acts before an error occurs, while a feedback system acts after an error is detected.

C. In a feedforward system, the control action depends on the output signal.

D. A feedback system cannot compensate for disturbances.

答案解析：
前馈系统（Feedforward）的特点是根据输入或已知干扰直接产生控制作用，力求在误差发生前就进行补偿。而反馈系统（Feedback）则是测量系统输出，将其与期望值比较，根据产生的误差进行纠正。因此，选项B“前馈系统在误差发生前行动，反馈系统在检测到误差后行动”的描述是正确的。

选项A描述的是反馈系统的典型行为。

选项C是错误的，前馈系统的控制动作依赖于输入或干扰信号，而非输出信号。

选项D是错误的，反馈系统的主要优势之一正是能够补偿未知或未建模的干扰。
