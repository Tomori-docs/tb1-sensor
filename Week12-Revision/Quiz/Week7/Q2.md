## ⚙️ 执行器特性选择解释

这个任务是要求根据给定的关键参数（**低分数行程**、**高效率**、**最小速度**），选择最符合这些参数的执行器类型。

下方表格提供了该问题的**答案**，以及一个行程的定义。

| 执行器类型 | 低分数行程 (Low fractional stroke) | 高效率 (High efficiency) | 最小速度 (Minimal speed) |
| :--- | :---: | :---: | :---: |
| **静电执行器 (Electrostatic actuator)** | | $\text{X}$ | |
| **热执行器 (Thermal actuator)** | | | $\text{X}$ |
| **压电执行器 (Piezoelectric actuator)** | $\text{X}$ | | |

---

### **关键参数与执行器匹配的原理说明**

1.  **低分数行程 (Low fractional stroke) $\rightarrow$ 压电执行器 (Piezoelectric actuator)**
    * **分数行程**指的是输出的行程与执行器长度的比值。压电材料的应变（形变）通常非常小（例如，在 $10^{-3}$ 到 $10^{-4}$ 数量级），因此它们产生的行程**绝对值小**，分数行程也**非常低**。
2.  **高效率 (High efficiency) $\rightarrow$ 静电执行器 (Electrostatic actuator)**
    * 静电执行器是基于电场力工作的，其原理是在两个电极之间施加电压来产生吸引力。理论上，静电场不消耗能量（**没有电阻损耗**），因此其驱动方式具有**极高**的能量转换效率。
3.  **最小速度 (Minimal speed) $\rightarrow$ 热执行器 (Thermal actuator)**
    * 热执行器通过材料的**热膨胀**或**相变**来产生形变和运动。热能的传递和材料的升温过程相对较慢，受到热容和散热的限制，因此其响应速度（即运行速度）通常是这三者中最**慢**的。

---

### **备注**

* **行程定义:** $Stroke = strain \times displacement$
    * 这个公式不太规范，但根据上下文，“位移”可能是指执行器的**有效长度**。
    * **规范的行程定义：** 行程（$\text{Stroke}$）应等于应变（$\text{Strain}$，即 $\Delta L/L$）乘以执行器的**长度**（$\text{L}$），即：$$\text{Stroke} = \text{Strain} \times \text{Length} = \frac{\Delta L}{L} \times L = \Delta L$$
